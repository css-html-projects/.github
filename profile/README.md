# Frontend Projects

A curated collection of frontend projects built to demonstrate practical skills in modern web development.

##  Overview

The repository includes multiple independent projects such as web applications, UI components, and landing pages developed using HTML, CSS, and JavaScript. 
Each project focuses on core frontend concepts including responsive design, layout systems, component structuring, and user interaction.

## Projects Included

* **expense-tracker** — a web application for tracking income and expenses with local storage
* **rock-paper-scissors** — a simple interactive game built with JavaScript
* **landing-mate-academy** — a responsive landing page created during a layout course
* **green-nest** — a landing page template for an online plant store
* **youtube-clone** — a static clone of YouTube interface for practicing HTML/CSS

## Tech Stack

* HTML5
* CSS3 (Flexbox, Grid, responsive design)
* JavaScript (ES6+)

### This collection is continuously growing as new projects are added and existing ones are improved.
